Propre
======

Introduction
------------

Propre is a handy tool written in Ruby to bulk rename your movies using TheMovieDB API.

Usage
-----

```
Usage: propre [OPTION]... SOURCE...
    -R, --recursive                  Run recursively
    -i, --interactive                Run interactively
```
